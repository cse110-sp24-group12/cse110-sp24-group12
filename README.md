## Instructions to run this project locally (for developers)

1. Clone the repo:

```sh
git clone https://github.com/cse110-sp24-group12/cse110-sp24-group12
```

2. install the required packages in the `source` directory

```sh
cd source
npm install
```

3. Run electron

```sh
npm start
```
